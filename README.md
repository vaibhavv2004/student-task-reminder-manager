# student-task-reminder-manager
A product management project demonstrating requirements gathering, user story creation, backlog prioritization, and sprint planning using Jira. Designed to help students track assignment deadlines and manage exam reminders effectively.

## Problem Statement
Students struggle to track multiple assignment deadlines across courses, leading to missed submissions and poor time management.

## Target Users
College/university students managing 4-6 courses simultaneously with varying assignment types and deadlines.

## Key Deliverables
- **Product Requirement Document (PRD)** - Defining problem, solution, and success metrics
- **5 User Stories** with detailed acceptance criteria
- **Epic & Backlog Management** - Prioritized using MVP framework
- **Sprint Planning** - 2-week sprint with 3 high-priority features
- **Agile Workflow** - Tracked in Jira using Scrum methodology

## Skills
- Requirements gathering and documentation
- User story writing with acceptance criteria
- Backlog prioritization (MVP vs Enhancement)
- Sprint planning and execution
- Jira proficiency (Scrum board, backlog, workflows)
- Agile/Scrum methodology

## Tools Used
- **Jira** - Project management and tracking
- **Google Docs** - PRD documentation

- Detailed user stories with acceptance criteria
